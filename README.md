# -BTC-Telegram-Simple-Crypto-Scraper-
A very simple scraper to collect BTC addresses from a nominated Telegram channel. 

This tool is a basic scraping tool to collect BTC addresses and is one of my first 'proper' Python OSINT projects, so it's very much a WIP and will be updated with improvements as and when. 

Change target_group to reflect what channel you want to scrape, and the limit to reflect how many past messages to target.

*** Do not share your API details  ***
